# LOF 套利领域知识

## 什么是 LOF？

LOF（Listed Open-Ended Fund）= 上市开放式基金，同时在两个市场交易：

| 市场 | 交易方式 | 价格 |
|------|---------|------|
| 一级市场 | 按净值申购/赎回 | 每日一个价（净值） |
| 二级市场 | 像股票一样买卖 | 实时变动（供需决定） |

两个价格不一样时产生套利机会。

## 溢价套利流程（核心玩法）

**条件**：场内价格 > 基金净值（溢价）

```
T日   → 按净值在一级市场「申购」基金
T+2日 → 份额到账（QDII 为 T+3）
T+2日 → 在二级市场按市场价「卖出」→ 赚差价
```

**利润 = 卖出价 - 申购净值 - 各项成本**

## 折价套利（较少使用）

**条件**：场内价格 < 基金净值

```
T日 → 二级市场按市场价「买入」
T日 → 申请「赎回」（按净值拿回现金）
```

⚠️ 持有不足 7 天赎回费 1.5%，实际很少操作。

## 成本结构

| 成本项 | 费率 | 说明 |
|--------|------|------|
| 申购费 | 0.15% | 券商渠道优惠后的常见费率 |
| 卖出佣金 | 0.025% | 券商佣金 |
| 冲击成本 | 0.1% | 卖出时的价格滑点 |
| 溢价衰减 | 1~2% | 最大不确定性，详见下方 |

### 溢价衰减

从申购到卖出等待期间，溢价率通常会下降：
- 国内基金（T+2）：预估衰减 1%
- QDII 基金（T+3）：预估衰减 2%（时间更长、不确定性更大）

衰减原因：
1. 基金公司降低限额或暂停申购
2. 市场情绪变化，溢价自然回落
3. 大量套利资金涌入，份额上市后砸盘

## 盈亏平衡点

```
国内基金：溢价率 > 1.275% 才有利润
QDII基金：溢价率 > 2.275% 才有利润
```

## 限购机制

高溢价品种几乎必然被限购：
- 溢价 > 5% → 大概率限购 5000 或更低
- 溢价 > 10% → 可能限购 1000 甚至暂停
- 限购随时可能调整

"一拖六"策略：用自己 + 家人共 6 个账户同时申购，放大受限购的收益。

## 风险因素

1. **时间风险**：T+2/T+3 期间溢价可能消失甚至反转
2. **流动性风险**：日成交额太低可能卖不出
3. **暂停申购风险**：高溢价品种随时可能被暂停
4. **极端溢价风险**：溢价率 > 20% 的品种通常不可持续

## 最佳实践

1. 每天 14:00~15:00 查看（尾盘数据最稳定）
2. 只操作 A 级推荐的品种
3. 多账户策略放大收益
4. 单账户收益 < 50 元不值得操作
5. 不要借钱做套利（本质是小钱低风险策略）
6. 关注基金公司公告（限额调整、暂停申购）
